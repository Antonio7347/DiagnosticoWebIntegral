# Diagnostico Web Integral

## Nombre del proyecto
Control de Estudiantes - Diagnóstico Web Integral

## Descripción
Aplicación web CRUD para gestionar estudiantes. Permite crear, consultar, editar y eliminar registros con al menos cuatro campos: nombre, edad, grado y correo.

## Tecnologías utilizadas
- Backend: Spring Boot, Spring Data JPA, H2 Database
- Frontend: React, Vite
- Lenguajes: Java, JavaScript, HTML, CSS

## Funcionalidades
- Registrar nuevos estudiantes
- Consultar lista de estudiantes
- Editar estudiante existente
- Eliminar estudiante
- Almacena datos en memoria usando H2 durante la ejecución

## Instrucciones para ejecutar el proyecto
1. Abrir el proyecto en un editor.
2. Iniciar el backend con Maven:
   - En la carpeta raíz del proyecto ejecutar `./mvnw spring-boot:run` o `mvnw.cmd spring-boot:run` en Windows.
3. Iniciar el frontend:
   - Ir a `DiagnosticoWebIntegralFront`
   - Ejecutar `npm install` si aún no está instalado
   - Ejecutar `npm run dev`
4. Abrir el navegador en `http://localhost:5173`
